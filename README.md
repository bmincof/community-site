Spring 공부 목적으로 커뮤니티 게시판 사이트를 구현했습니다. 회원기능, 게시글, 댓글, 추천 등의 간단한 기능을 제공합니다.

## 개발 기간

2022.08.03 ~ 2022.10.29 (약 3개월)

프레임워크 : Spring Framework 5.2.19
